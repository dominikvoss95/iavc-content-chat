# IAVC World Content Chat

A Python-based content chat agent using LangGraph, FastAPI, and httpx to parse, index, and query IAVC World articles.

## Setup
1. Create a virtual environment: `python3 -m venv venv`
2. Activate it: `source venv/bin/activate` 
3. Install reqs: `pip install -r requirements.txt` (or manually the ones listed in the setup)

## Run Server
```bash
python -m src.main
```

## Running Tests
```bash
pytest
```
